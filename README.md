# GD SFX

(we should find a better name for this project)

GD SFX is a Geometry Dash SFX Library explorer.

## Features

- [x] SFX Playback
- [x] Categories
- [x] Download and delete
- [x] Favourites
- [x] Searching
- [x] Sort by name / duration / id / size
- [x] SFX Credits
- [x] Statistics
- [x] Settings
- [x] Mass download and mass delete
- [ ] Change speed and pitch
- [ ] Change volume
